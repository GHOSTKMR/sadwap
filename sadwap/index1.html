Today's Knowledge Tips

KDS System Restart Command

Use this command to restart all available KDS screens in operations:

c:/bootdrive/aloha kitchen/data/currentdate/ctrl+a;delete;ctrl+s

This command will clear the current date's data and restart all KDS screens.
